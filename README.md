# Revenue AI Nexus

This project is an AI-powered revenue management platform.

## Troubleshooting Filesystem Errors
If you encounter errors like `ENOTEMPTY` or `rmdir` failures:
1. **Restart the Dyad App**: This is the most common fix for locked files.
2. **Restart your Computer**: This ensures all background processes (like Node.js or Vite) are fully stopped.
3. **Try Restart/Rebuild**: Once back in, use the action buttons above the chat to start the server.
